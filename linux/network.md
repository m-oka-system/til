## IP アドレス設定

```bash
# IPアドレスの確認
ip a

# IPアドレスの設定ファイル
/etc/sysconfig/network-scripts/ifcfg-xxx

DEVICE=eth0              # デバイス名
BOOTPROTO=dhcp           # DHCPを使う
ONBOOT=yes               # 起動時に有効にする
TYPE=Ethernet            # タイプ
IPADDR=10.0.11.10        # IP
NETMASK=255.255.255.0    # SubnetMask
NETWORK=10.0.11.0        # ネットワークアドレス
HWADDR=06:c6:e1:0c:27:3a # MacAddress

# IPアドレス変更を反映
systemctrl restart network.service
```

## DNS 設定

```bash
# /etc/nsswitch.conf
# hosts -> dnsの優先順位
hosts:      files dns myhostname

# /etc/hosts
127.0.0.1   localhost localhost.localdomain localhost4 localhost4.localdomain4
::1         localhost6 localhost6.localdomain6

# EC2の/etc/resolve.conf
options timeout:2 attempts:5
; generated by /usr/sbin/dhclient-script
search ap-northeast-1.compute.internal
nameserver 10.0.0.2
```

## TCP サービスの確認

```bash
# TCPサービスを表示
ss -at

a:display all sockets
t:display only TCP sockets
u:display only UDP sockets
```
